## الگوریتم FCFS چیست؟

**FCFS | First‑Come, First‑Served** پردازش‌ها را دقیقاً به ترتیب ورود اجرا می‌کند.

## C کد جاوااسکریپت

<CodeBlock
  codeInstances={[
    {
      code: `function fcfsSchedule(processes) {
  processes.sort((a, b) => a.arrivalTime - b.arrivalTime);

  let time = 0;
  return processes.map(p => {
    if (time < p.arrivalTime) time = p.arrivalTime;
    const start = time;
    time += p.burstTime;
    return { ...p, start, end: time };
  });
}`,

## D جمع‌بندی

FCFS به دلیل **سادگی** کاربرد آموزشی زیادی دارد، ولی در بارهای متنوع ممکن است زمان انتظار را افزایش دهد.
